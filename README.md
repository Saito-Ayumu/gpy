# gpa.py

成績(S,A,B,C,D)を入力しGPAを計算します。

---

## 使い方

```bash
$ echo "S A B C D" | ./gpa.py
GPAは2.00 です
```

---

## 成績からのGPA算出

* 入力: 成績"S A B C D"(空白区切り)
* 出力: GPAを1行で出力
* 無効な成績:無視

---

## インストール
```bash
git clone https://github.com/Saito-Ayumu/gpa.py
cd gpa.py
chmod +x gpa.py test.bash
```
